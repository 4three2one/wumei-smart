- java目录 ------- 放置后端打包好的server.jar文件，安装时文件上传到服务器或本地的 /var/data/java 目录中
- nginx目录 ------ 放置前端打包好的文件，html文件夹上传到服务器或本地的 /var/data/nginx 目录中
- emqx目录 ------- 放置emqx配置文件
- mysql目录 ------ 放置mysql配置文件和初始化的数据脚本
- redis目录 ------ 放置redis的持久化数据，运行时自动生成
